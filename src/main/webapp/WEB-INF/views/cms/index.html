2016年3月16日 星期三

站内公告

各县（市）区委宣传部，各县（市）区司法局、妇联，市各有关单位： 为深入贯彻党的十八届四中全会精神，加强家庭法治宣传教育，提高广大家庭成员的学法、用法意识，促进家庭和谐幸福，推进全市“六五”普法，决定在全市举办“家庭学法达人赛”，普及婚姻家庭相关法律知识，掀起家庭学法热潮。现将有关事项通知如下： 一、参赛对象：全市广大家庭成员。 二、活动时间：2015年9月28日―10月27日。 三、竞赛内容：《婚姻法》、《继承法》、《妇女权益保障法》、《未成年人保护法》、《老年人权益保障法》等与家庭生活相关的法律法规。 四、活动平台：南通宣传网（http://xcw.nantong.gov.cn）、南通普法网（www.ntsfj.gov.cn）、南通女性网（www.ntwomen.org）、南通网（http://www.zgnt.net）、江海明珠网（www.ntjoy.com）、南通文明网（www.ntwenming.com）、濠滨论坛（http://bbs.0513.org）。 五、参与方式：登录以上网站，点击“全市家庭学法达人赛”，或者直接登录网址http://58.221.162.245/xfdr/进入“学法达人”网站，注册并登录后，点击“在线对决”，选择“全市家庭学法达人赛”活动名称，进入系统进行答题。注册登录手机号码和姓名必须对应，否则一旦中奖无法充值；其他信息须填写准确，以便主办方准确统计评选优秀组织奖。 五、奖励办法：竞赛设立优秀个人奖和优秀组织奖。优秀个人奖100名，根据个人得分确定获奖名单，如果获得最高分的人数多于奖项名额，则采取随机方式抽取获奖者；优秀组织奖若干个，根据各地区、各部门参赛的组织程度确定获奖名单。活动结束后将在活动网站公布获奖名单，优秀个人奖奖金为每人100元话费充值。 六、组织要求：各地区、各部门要高度重视，认真组织，广泛发动，引导广大家庭积极参与，促进婚姻家庭相关法律知识在家庭中的学习和普及。各县（市）区相关部门要整体联动，层层发动，增强活动影响力。市各有关单位要积极组织发动本部门、本系统干部职工参加知识竞赛，尽可能扩大参与面。 中共南通市委宣传部 南通市司法局 南通市妇女联合会
